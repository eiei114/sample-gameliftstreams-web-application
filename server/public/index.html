How to use
Shortcuts
Hide/Show metrics widgets: CTRL-S
Exit full screen: Press ESC
Test stream session
Option 1: Use the Stream Demo Configuration UI when loading index.html. Add a User ID, Stream Group ID, and Application ID, then click to start.

Option 2: Use a URL: http://{End Point URL}/?userId=Player1&streamGroupId={Stream Group ID}&applicationId={Application ID}
Export files action
When ending the game session, you have the option to view the full game session metrics and download metrics data.
Troubleshooting tips
For mobile testing, use the stream URL on a mobile device or use Chrome browser's development tools with the device toolbar.
Attaching and detaching inputs
Once your Game Stream opens, you will see an "Attach Input" option in the top right menu. This will attach mouse, game controller, and other inputs to the focused active stream video element window. When the video element window is out of focus, you can still interact with the browser and menu options.
Fullscreen
This will make the active stream video full screen. To exit full screen, press the Esc key on your keyboard.
Close Connection
Click the Terminate menu button. This will bring you to an End Game Session menu.
- End Game Session button: This will end the current stream and game session.
- End Session / View Metrics: This will end the current stream and game session and show the full game session metrics screen.
Metrics
Click the Metrics menu button to enable metrics recording. Use the Metrics menu gear icon to add and remove metrics widgets. You can move widgets around the screen and view them in full screen mode. To toggle them on and off, use CTRL-S.
Next steps
Explore different game settings, try various input devices, and analyze the metrics to optimize your streaming experience.
